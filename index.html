Tentang

Halo, nama saya Dian Mustofa. Saya memiliki minat yang besar terhadap teknologi. Ketertarikan saya berawal dari menonton video YouTube tentang berbagai kreasi menggunakan Arduino Uno. Dari situ, saya mulai belajar membuat proyek sederhana seperti jam running text, sensor jarak, sensor suhu, dan lainnya. Ketertarikan saya kemudian berkembang ke dunia IoT (Internet of Things), di mana saya belajar menampilkan data sensor secara realtime ke dalam sebuah website. IoT membuka wawasan saya bahwa teknologi bisa digunakan untuk membuat proyek-proyek canggih seperti smart home dan kebun pintar. Dari situ pula saya mulai belajar tampilan website sebagai antarmuka untuk menampilkan data dari sensor.
